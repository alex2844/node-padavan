# Конфигурация


### Принимает

| credentials   | Описание
| ---           | ---
| *`repo`*      | Репозиторий где собираются прошивки
| *`branch`*    | Ветка для устройства
| *`token`*     | Токен для доступа к github
| `host`        | IP роутера
| `username`    | Логин для доступа к роутеру
| `password`    | Пароль для доступа к роутеру
