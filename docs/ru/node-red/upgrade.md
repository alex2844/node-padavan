# Обновления

Управление прошивками


### Принимает

| msg       | type
| ---       | ---
| `topic`   | string


| topic         | Описание
| ---           | ---
| *`build`*     | Собрать прошивку
| *`changelog`* | Список изменений
| *`upgrade`*   | Установить прошивку


### Выводит

| msg       | type      | Значение
| ---       | ---       | ---
| `topic`   | string    | changelog
| `error`   | string    |
| `payload` | object    |


| payload   | Описание
| ---       | ---
| `from_id` | Текущая прошивка
| `to_id`   | Собранная прошивка
| `data`    | Массив с изменениями
